# Oh, the places you'll go! 

A simple app to search the google places library and display results!
## Usage
### Run Locally

`npm install`

### Serve your files on a webpack server 

`npm start`

Then go to localhost:8080.

### For Development to hotload file changes

`webpack`

`webpack --watch`
